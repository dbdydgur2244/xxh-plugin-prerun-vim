Vim plugin for xxh bash.

## Install
From xxh repo:
```
xxhp i xxh-plugin-bash-vim
xxh yourhost +s bash-zero +if
```
From any repo:
```
cd ~/.xxh/xxh/plugins \
    && git clone --depth 1 https://github.com/xxh/xxh-plugin-bash-vim \
    && ./xxh-plugin-bash-vim/build.xsh
    
# connect once with updating host
xxh yourhost +s bash-zero +if
```
